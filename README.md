# > devroast

Cole seu código. Receba um veredito.

**devroast** analisa trechos de código via IA e devolve uma nota de 0 a 10, junto com um breakdown das issues encontradas — organizadas por severidade. Sem eufemismos, sem padrão corporativo. O feedback é direto ao ponto.

O projeto tem um leaderboard público onde os piores códigos submetidos ficam ranqueados. A ideia é simples: quanto pior o código, maior a exposição. Incentivo passivo para escrever código melhor.

---

**roast mode** ativo por padrão. Desative se quiser um tom mais diplomático — mas ninguém vai te julgar se deixar ligado.
